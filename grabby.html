Grabby

Shareable Grocery Lists
A simple, collaborative grocery list app built with React Native and Expo. Create your grocery list, share it with family and friends using a simple share code, and shop together seamlessly.
Create and manage your grocery list with easy add, check, and delete functions
Share your list instantly with a 6-character code that anyone can use to join
Collaborate in real-time with family and friends on the same shopping list
Development Status
Grabby is currently in development as a React Native mobile app. The core features are implemented including list creation, sharing via codes, and real-time collaboration through Supabase. Stay tuned for the official release!
